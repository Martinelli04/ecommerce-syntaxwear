
# SyntaxWear - Tênis e Sneakers Online

## Descrição

SyntaxWear é um e-commerce de tênis e sneakers online. A plataforma foi desenvolvida para proporcionar uma experiência de compra fácil e agradável, oferecendo uma variedade de modelos para todos os estilos.

## Funcionalidades

- **Navegação Intuitiva:** Menus para categorias masculino, feminino e outlet.
- **Busca Rápida:** Acesso rápido a lojas, informações sobre a empresa, conta de usuário, ajuda e carrinho de compras.
- **Seção de Destaque (Hero):** Apresenta os principais produtos e promoções.
- **Categorias de Produtos:** Seções dedicadas para tênis casuais, esportivos, modernos e futuristas.
- **Grid de Produtos:** Exibição organizada dos produtos com destaque para modelos especiais.
- **Newsletter:** Formulário para inscrição e recebimento de novidades por e-mail.
- **Rodapé Completo:** Links para redes sociais, navegação adicional e informações de copyright.

## Tecnologias Utilizadas

- **HTML5:** Estrutura semântica da página.
- **CSS3:** Estilização e design, com uma abordagem modular e organizada:
  - `reset.css`: Normalização dos estilos padrão dos navegadores.
  - `variables.css`: Definição de variáveis de cores, fontes e outras propriedades.
  - `base.css`: Estilos básicos aplicados a todo o site.
  - `components/`: Estilos específicos para cada componente da página (header, footer, hero, etc.).

## Como Utilizar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Martinelli04/ecommerce-syntaxwear.git
   ```
2. **Navegue até o diretório do projeto:**
   ```bash
   cd ecommerce-syntaxwear
   ```
3. **Abra o arquivo `index.html` no seu navegador de preferência.**

   Você pode fazer isso clicando duas vezes no arquivo `index.html` ou usando um servidor local para visualizar o site.
